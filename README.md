# Stock Price Forecasting using ARIMA

## Introduction
This project focuses on predicting future stock prices using Time Series Forecasting techniques. The ARIMA model is used to analyze historical stock data and forecast future values.

## Data Source
The stock price data is collected from Yahoo Finance using the yfinance Python library.

## Tools and Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Statsmodels
- yfinance
- Google Colab
- GitHub

## Steps Involved
1. Importing libraries
2. Collecting stock price data
3. Data preprocessing
4. Data visualization
5. Calculating Moving Average
6. Building ARIMA model
7. Forecasting future stock prices
8. Plotting forecast results

## Model Used
ARIMA (Auto Regressive Integrated Moving Average) Time Series Model

## Results
The model successfully forecasts future stock prices based on historical trends and patterns.

## Conclusion
Time series forecasting is useful for analyzing stock trends and predicting future prices. ARIMA model provides a good baseline model for forecasting financial data.
